# Web-Development
SCU coursework for CSEN 161 lab (Web Development lab)

The content for this course is as follows:

Lab 1: Use of basic html to layout the format of all web pages.

Lab 2: Added CSS for styling the different html web pages.

Lab 3: Implemented highlighting functionality and saving user position in article.html using JavaScript. Used JSON to store article data.

Lab 4: Added PHP to the website to display articles dynamically from the articles.json data.

Lab 5: Used SQLite to store and query for articles rather than using json.
